Data from an Excel File

The Excel file must obey following rules:

1) Latitude and longitude must be written in two separate columns (as dd°mm.mmm', dd°mm'ss.s'', Gauss-Krüger oder UTM).

2) Azimuth and angle of dip may be listed together with a separator in one column or in two separate columns.

3) The value which is written in the same row as the coordinates is plotted.

4) Other text written in these columns is overread.

Procedure:

1) Find the folder.

2) Click on the Excel file.

3) Choose the worksheet.

4) The Excel worksheet is displayed: find the colums for the coordinates and for azimuths and dip angles. Minimize the worksheet.

5) Type the columns for latitude and longitude.

6) Type the columns for azimuth and dip angle.

7) Choose the fabric type.

8) Type the numbers of the first and last row of the choosen area of the worksheet.

9) Type the symbol of the separator if azimuth and dip angle are written in the same column.

10) Click on 'OK'.
