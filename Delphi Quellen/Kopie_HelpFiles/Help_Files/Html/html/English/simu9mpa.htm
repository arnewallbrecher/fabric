Plot Area for Simulation of Orientation Data

An empty diagram appears in the plot area. In order to produce simulated data you have to choose the projection you want to use for the diagram and the fabric type (planes, linears, etc.). Click on the places for which you want to produce data. Data points appear in the colour chosen in the menu colour. The readings appear in the data list to the right of the plot area.

After pressing the button 'finish simulation' the data produced can be saved (menu 'Save') and/or printed (menu 'print').
