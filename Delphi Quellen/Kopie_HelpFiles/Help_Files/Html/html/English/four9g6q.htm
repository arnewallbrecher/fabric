Fourier-Analysis of Fold Shapes

This program calculates the Fourier-coefficients from the shape of a fold (RAMSAY & HUBER, 1987:314ff.). The profile of a fold is regarded as a harmonic wave which can be decomposed into its harmonic components by Fourier analysis. The general formula for description of an arbitrary harmonic wave with Fourier coefficients is

f(xy) = a0 + a1cos x + a2cos 2x + a3cos 3x + .......

+ b1sin x + b2sin 2x + b3sin 3x + .......

For the description of the profile of a tectonic fold the number of coefficients reduces to a maximum of three. The reasons are:

1. The profile can always begin at the inflection point. If we put this point in the zero point of the coordinate system, a0 and all cosine terms become 0.

2. Assuming that the fold is symmetrical, we only look at a quarter fold (i. e. the profile between inflection and hinge) in this case all terms with even indices become 0.

3. For a sufficiently exact description of the fold shape we only need the first three coefficients.

b1, b3 und b5.
