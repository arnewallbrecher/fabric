The File Licence.bin‚

This file contains the name of the licence owner when Fabric7 has been bought for unlimited usage. When Fabric7 is only used for one month testing, the word 'Demo' is used. There is a special licence for students of earth sciences at Karl-Franzens-Universität Graz. This licence does not allow distribution of the program to other persons.

The name of the licence owner appears on the start page.
