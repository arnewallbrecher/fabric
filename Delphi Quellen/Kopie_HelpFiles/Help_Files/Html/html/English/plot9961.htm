Plot into a HPGL-File:

With this option a *.plt-file can be created which can produce a vector plot. Such a file can be imported to Corel Draw or othe plot programs. This option opens a panel on the right hand of the plot surface for input of the radius of the diagram and the coordinates of the centre.

The readings from the last Fabric7 application are saved in Fabric7.ini and may be changed. With ‚OK‚ printing is started.
