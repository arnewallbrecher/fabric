Determination of the Shape Factor of a Paleo Stress-Ellipsoid

This Program allows the estimation of the shape factor R according to WALLBRECHER et al. (1996). Using Bott's formula (BOTT, 1959) the maximum resolved shear stresses (MRSS) i.e. directions of striation on the measured slickenside planes are calculated for R = 0.0, 0.1, 0.2, ...,1.0.

Eigenvalues are calculated from the theoretical striation populations. These Eigenvalues are shown in a triangle plot. The eigenvalues of the measured striations are plotted as well. By visual interpolation the R-value can be estimated. The program needs the principal stress axes, i.e. the data set has to be treated with the dihedra method before running this program. It is advisable to check the consistency of the data set in a Hoeppener-diagram first, in order to eliminate non-fitting data.
