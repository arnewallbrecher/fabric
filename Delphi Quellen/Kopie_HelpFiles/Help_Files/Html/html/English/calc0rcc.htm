Manual Data Input:

The orientation of planes or linears is described by azimuth of dip and angle of dip. You have to fill in the orientations of the two fabric elements which shall be used in the calculation. Clicking on 'OK' or pressing the return key start the calculation. Input data and calculated fabric element appear in the list field.

After pressing the 'terminate input' button, the input data and/or the results can be saved or printed.

Maximum number of data indicates the number of data sets which can be calculated.
