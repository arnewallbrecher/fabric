Vertrauenskegel

Die Zuverlässigkeit des errechneten Schwerpunktvektors wird durch einen Kegel mit dem Schwerpunktvektors als Kegelachse dargestellt. In der Projektion ist das ein Kleinkreis. Dieser Kegel kann für verschiedene Vertrauensmaße (Konfidenz, Irrtumsschwelle) berechnet werden. Drei Vertrauensmaße können im Menu ‚Konfidenz festgelegt werden: (90%, 95% und 99%). Ein Vertrauensmaß von 99% bedeutet, dass ein

Prozent der Daten außerhalb dieses Kegels liegen kann. Der Öffnungswinkel des Kegels (Θ) beträgt:


n = Zahl der Daten

p := Irrtumsschwelle (Konfidenz, Vertrauensintervall) vorgegeben sind die Werte 0.1, 0.05 oder 0.01.

Der Vertrauenskegel wird als Kleinkreis um den Schwerpunktvektor gezeichnet. Seine Farbe wird in Fabric7.ini gespeichert und kann im Menu ‚Farbe des Vertrauenskegels' geändert werden.
