Plot von Isolinien

Die Abstufung der Isolinien kann in Vielfachen der Gleichverteilung mit logarithmischer Abstufung oder mit benutzerdefinierter Abstufung vorgenommen werden. Im ersten Fall werden fünf Isolinien ausgerechnet und automatisch geplottet. Die Isolinien sind logarithmisch zwischen 1 und der maximalen Besetzungsdichte abgestuft. Die Zählkalotte wird der Datenmenge angepaßt, sodaß Vielfache der Gleichverteilung durch die Isolinien abgegrenzt werden. Im zweiten Fall kann die Abstufung der fünf Isolinien manuell eingegeben werden. in diesem Fall wird auch die Größe der Auszählkalotte frei gewählt. Die gewählte Methode wird in Fabric7.ini gespeichert.

Im Menupunkt Farben\Isolinien können auch Grauwerte für die Darstellung der Isolinien gewählt werden.

Im Menu ‚Linien‚ wird der Linientyp für jede Isolinie festgelegt.

Mit dem Menu ‚Konturierung‚hat man weitere Methoden zur Darstellung der Belegungsdichten zur Verfügung

Das Menu ‚Glättungsgrad wird nur bei der Skalarprodukt-Methode aktiv.

Das Menu ‚Graphik‚ ist das gleiche wie bei den vorigen Programmen.
