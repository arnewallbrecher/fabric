Cluster von Richtungsdaten, Plot

Die signifikante Trennung von Verteilungshäufungen (Clustern) wird vorgenommen.

Für die gefundenen Cluster werden die statistischen Parameter wie in Programm 8 (Gefuege) berechnet. Spärischer Öffnungsgrad und Vertrauenskegel werden um den Cluster-Schwerpunkt (Schwerpunktvektor)gezeichnet. Im Menu Konfidenz kann das in Fabric7.ini gespeicherte Vertrauensintervall geändert werden.

Im Menu Farbekönnen die in Fabric7.ini gespeicherten Farben aller graphisch dargestellten Parameter geändert werden. Im Menu ‚Datenpunkte‚ kann das Symbol für die Datenpunkte geändert werden.

Das Menu ‚Graphik‚ ist das gleiche wie bei den vorigen Programmen.
