Rekonstruktion

Mit dieser Option kann durch Anklicken der Datenpunkte in einem Lagenkugel-Diagramm wieder eine Datenliste erzeugt werden. Dies ermöglicht z.B. die statistische Auswertung des Diagrammes mit vektorstatistischen Methoden.

Von dem Diagramm, dessen Daten rekonstruiert werden sollen, wird ein Bitmap (z.B. durch Einscannen) erstellt. Mit einem Grafik-Programm (z.B. Irfanview oder Corel-Paint) wird das Bitmap so beschnitten, dass die Markierungen genau auf die Peripherie fallen. Der Ausschnitt wird in die Zwischenablage gebracht.

Nach Klicken auf ‚Rekonstruktion‚ und Markieren des ok-Feldes erscheint das Plotfeld mit dem Bitmap.
