Extension:

Fabric7 uses the following extensions for file names:

*.AZI = Directions with polarity but no magnitude on a horizontal plane (0..359°)
*.FOU = Fourier-Coefficients
*.FRN = normierte Fry-Methode
*.FRY = Koordinaten-Matrix für Strain-Analyse mit der Fry-Methode
*.LAZ = Directions with polarity and magnitude on a horizontal plane
*.LI2 = Pairs of linear data for the calculation of planes, angles, or normals
*.LIN = 3-dimensional lineation data (azimuth of dip / angle of dip)
*.LST = Directions with magnitude but without polarity on a horizontal plane (type foto lineations).
*.PL2 = Pairs of planes for calculation of intersection lines or angles.
*.PLA = 3-dimensional data of planes (azimuth of dip / angle of dip)
*.RGD = Data for a R%-G%-diagram
*.SLI = Slickenside planes with striations with polarity
*.STR = Directions without polarity and magnitude on a horizontal plane (strike lines)
*TAL = Distants of isogons for a Talpha'/Alpha-diagram
*.WOO = Eigenvalues
