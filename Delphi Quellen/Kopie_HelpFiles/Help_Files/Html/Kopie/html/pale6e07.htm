Input of Displacement Data from a File

After pressing the button 'Start' the list field on the right side ('select files') shows the content of the directory whose path is saved in Fabric7.ini. If this field is empty and the message 'no file found' appears, the data path can be changed after pressing the button 'find data path'.

Click on the files you want to use. These files appear in the middle field ('selected files'). Files can be removed from this list by clicking on them.

The button 'selection ok' terminates the choice of files.

The button 'plot' opens the plot form. The data of the files used appears in the left field.

The extension of the files and the path chosen appear under the left list.
