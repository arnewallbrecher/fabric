Shape Factor

The shape factor (R) is the a ratio derived from the principal shear stresses. It describes the shape of the stress-ellipsoid.

The meaning of the shape factor can be understood if you consider the ratio of the three unscaled Mohr-circles:

R= 0.0, if σ2 = σ3;

R = 0.5, if σ2 = (σ1 – σ3)/2;

R = 1.0, if σ2 = σ1.
