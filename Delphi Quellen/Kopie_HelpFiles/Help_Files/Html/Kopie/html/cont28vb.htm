Smoothing Factor:

With an increasing smoothing factor you improve the smoothness of the contour lines. This goes together with a loss of details. You have to find the optimum by comparison with a diagram plotted with the counting calotte method. This comparison can easily be done by overlay with a counting calotte diagram. by applying the the options 'copy‚ and ‚load from clipboard‚ of the menu 'graphic'. This comparison with diagrams plotted with various smoothing factors lets you find the best compromise.

The smoothing factor is based on the influence of remote points. With an increasing smoothing factor the influence of a far away point becomes less important. This effect is reached by raising cosγ (s. scalar product method) to a higher power:

di = counting density of the i-th pixel,

γj = angle between the pixel (as projected point) and the j-th data point

n = number of data

g = smoothing factor
