Scalar Product Method:

This method uses each pixel of the diagram as counting point. The angle between the direction of this pixel and the direction of each data point is calculated and saved as cosine functions (scalar product). See Adam, 1989). The sum of these cosines represents the counting density of the pixel.

di = counting density of the i-th pixel,

γj = angle between the pixel (as projected point) and the j-th data point

n = number of data

a data point with γ = 0 encreases d by 1. A data point with mit γ = 90 encreases d by 0.

The method produces very smooth contour lines between the counting densities. The gradation can be in steps or floating.
