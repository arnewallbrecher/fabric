Reconstruction

This option enables you to reconstruct the original list of orientation data from the data points in an already existing data plot. This is useful if you want to calculate statistical parameters of an already existing diagram.

In a first step you have to produce a bitmap of the diagram for which you want to reconstruct the data (i.e. by scanning). With a plot program (i.e. Irfanview or Corel-Paint) you cut out the diagram (the cutting frame should touch the periphery of the projection). The cut out is saved on the clipboard.

After clicking on ‚Reconstruction‚ and on 'OK' the bitmap with the diagram appears in the plot area.
