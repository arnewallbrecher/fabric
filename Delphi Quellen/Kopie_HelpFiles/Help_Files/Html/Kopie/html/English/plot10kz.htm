Plot of Great Circles:

This program shows planes as great circles. The projections used in this program are the same as those in the previous program. The algorithsm for the plot is described by Wallbrecher, 1986:39ff.

In the menu 'data input' you choose the drive which contains the data or 'manual' if you want to add new data.

Manual input is also possible for universal stage data like azimuths and tilt angles of normals to planes (i.e. poles to mica-planes). These data are converted to azimuths of dip and angles of dip internally. All further processing of these data is done in this format.

The type of data must be set int the menu 'fabric type'. In the menu ‚angle‚ you choose input in the 1..360° or 1..400gon scale.

For data input from a file the fabric types "U-stage-axes" and "U-stage-poles“ are not active, because U-stage data are saved in *.LIN files.

Data input from an EXCEL-sheet is also possible. In the sheet azimutth of dip and angle of dip may either be written in a single row (with a separation key) or in two separate rows.
