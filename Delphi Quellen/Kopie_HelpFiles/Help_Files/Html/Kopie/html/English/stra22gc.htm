Strain Analysis with the Talbot Method, Manual Data Input

The manual input is the same as for orientation data. Compressional structures are written with negative azimuths of dip. The input of azimuth of dip and angle of dip is separated by 'enter'. Input of a fabric element is terminated by ‚OK‚ or ‚Enter‚. The input is shown in the table on the right side. The button 'terminate input' activates the menus 'save data' and 'print'. The button 'plo' opens the plot form.
