Plot Area for Reconstruction of Orientation Data

The bitmap appears in the plot area. For the reconstruction of data you have to select the projection with which the diagram was drawn and the fabric type (planes, linears, etc). Click on all data points. The points which were clicked on appear in the colour chosen in the menu Colour. and as readings in the data list to the right of the plot area.

After clicking the button 'finish reconstruction', the data list can be saved (menu 'save') and/or printed (menu 'print').
