The File ‚Fabric7.ini‚

This file contains all the settings used in the entire program (i.e. colours, paths, language, statistical parameters, contour lines, etc. You can open this file with a text editor and change the settings. Do not erase lines in this file!

We suggest being very careful with these changes. It is better to change settings during the runtime of the program. Changes of settings during runtime are saved in this file.
