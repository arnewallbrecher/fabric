Significant Separation:

The data which belong to one cluster are found by drawing a cone with the apical angle t around the direction of each data point. Data with overlapping cones belong to the same cluster. Cluster t is the test angle, it depends on the counting density, which is shown in multiples of even distribution.


The statistical parameters of each cluster are calculated. Significant separation is given when the cones of spherical aperture + cone of confidence do not overlap with those of other clusters. Overlapping clusters are put together to 'pooled groups. The statistical parameters if these pooled groups are then calculated.
