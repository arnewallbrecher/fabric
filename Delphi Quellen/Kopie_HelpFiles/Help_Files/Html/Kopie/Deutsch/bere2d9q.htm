Extension:

Fabric7 verwendet folgende Extensionen für Dateinamen:

*.AZI = flächig verteilte Richtungsdaten 0 bis 359 Grad Typ Azimut

*.BMP = Bitmaps von Falten für Isogonen-Methode oder von Texturen

für Strain-Analyse

*.FOU = Fourier-Koeffizienten Typ Vergenz,x1,y1 x2,y2 x3,y3

*.FRN = Normalized Fry method

*.JPG = JPG-Bilder von Falten für Isogonen-Methoden oder von

Texturen für Strain-Analyse

*.LAZ = polare flächig verteilte Richtungsdaten mit Beträgen

*.LIN = räumlich verteilte lineare Richtungsdaten Typ Fallazimut/Fallwinkel

*.Li2 = Datei mit Paaren von Linearen für die Berechnung von

Flächen, Winkeln oder Normalen

*.LST = bipolare flächig verteilte Richtungsdaten mit Beträgen

Typ Fotolineation

*.PLA = räumlich verteilte flächige Richtungsdaten

Typ Fallazimut/Fallwinkel

*.PL2 = Datei mit Flächenpaaren für Berechnung von Schnittkanten

oder Winkeln

*.POL = Polygon-Eckpunkte für die Panozzo-Methode

*.RGD = Daten für R%-G%-Diagramme Typ R%,G%

*.SLI = Harnischflächen mit polaren Striemungsrichtungen

Typ Fallazimut/Fallwinkel/Fallazimut/+ od. - Fallwinkel

*.STR = Flächig verteilte bipolare Richtungsdaten

Typ Streichlinien

*.TAL = Isogonenabstände für T‚alpha/Alpha

*.TLI = Lineare (Dehnung und Stauchung) für die Talbot-Methode

*.WOO = Datei mit Eigenwerten Typ E1,E2,E3
