Die Datei ‚Lizenz.bin‚

Die Datei ‚Lizenz.bin‚ enthält den Namen des Lizenznehmers, wenn Fabric7 zur unbefristeten Anwendung erworben wurde. Wenn das Programm zum Testen für einen Monat angefordert wurde, enthält sie das Wort ‚Demo‚. Für Studenten der Erdwissenschaften der Karl-Franzens-Universität Graz besteht eine besondere Lizenz. Diese besagt jedoch, daß das Programm nicht an andere Personen weitergegeben werden darf.

Der Name des Lizenznehmers erscheint auf der Startseite.
