Hoeppener-Diagramm

Der Flächenpol einer Harnischfläche wird als Berührungspunkt dieser Fläche mit der Lagenkugel betrachtet, d.h. die Fläche wird als Tangentialfläche an die Lagenkugel aufgefaßt. Die Striemungsrichtung wird als ein von diesem Punkt ausgehender Pfeil auf der Lagenkugel dargestellt, der die Gleitrichtung auf dem Liegendblock darstellt (HOEPPENER, 1955).
