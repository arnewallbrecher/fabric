Messdaten in eine Karte eintragen

Mit diesem Programm können können Meßdaten (Lineare oder Flächen) in eine Karte eingetragen werden. Die Dateneingabe kann auf folgende Weise erfolgen:

1)manuelle Eingabe

2)aus einer Datei

3)durch Einscannen einer Probenkarte

4)aus einer Excel-Datei

Das Programm benötigt die Koordinaten der Meßpunkte in geographischen Koordinaten (dd°mm.mmm' oder dd°mm'ss.s''), Gauss-Krüger-Koordinaten oder UTM-Koordinaten) und Azimut und Fallwinkel der Messungen, sowie die Angabe, ob es sich um eine Fläche oder ein Linear handelt.

Die Meßwerte können in Altgrad oder Neugrad eingegeben werden. Die Eingebe aus einer Datei erfolgt immer in Altgrad.

Die Sektoren (NE, SE, SW, NW) werden für die manuelle Eingabe und die Eingabe mit Karte für geographische Koordinaten benötigt. NE bedeutet nördliche Breite und östliche Länge, entsprechendes gilt für die anderen Sektoren.
