Strainanalyse mit der Fry-Methode

Dies Programm führt eine Strain-Analyse nach Fry (1979) durch und plottet ein Diagramm, mit dem die beste Anpassung an eine finite Strain-Ellipse gefunden werden kann (Fry-Methode).

Die Dateneingabe erfolgt entweder aus einem Bild, aus einer Koordinaten-Dateioder aus einer Polygon-Datei.

Die Daten können entweder für die konventionelle Fry-Methode oder für die normierte Fry-Methode erhoben werden.
