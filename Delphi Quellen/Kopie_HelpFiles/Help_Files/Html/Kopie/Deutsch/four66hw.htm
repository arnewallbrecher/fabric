Fourier-Analyse, Dateneingabe aus einem Bild

Die Koordinaten einer Viertelfalte können direkt aus einem Bild durch Anklicken ermittelt werden.

Klicken Sie auf 'Bild importieren' um eine *.BMP oder *.JPG-Datei zu öffnen. Die Grafik kann entweder aus der Zwischenablage ('einfügen') oder aus einer Datei geladen werden.

Im Menu 'Bild korrigieren' haben Sie die Möglichkeit, das Bild zu drehen oder eine Vergenz rückgängig zu machen.

Die Koordinaten der Viertelfalte werden folgendermaßen bestimmt: Klicken Sie nacheinander auf eine Faltenumbiegung, auf den Scheitelpunkt, dann auf die 1/3- und 2/3-Koordinate (X1 und X2).
