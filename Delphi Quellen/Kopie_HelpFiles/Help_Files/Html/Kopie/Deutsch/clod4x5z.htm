Signifikante Trennung

Die zu einem Cluster gehörenden Daten werden gefunden, indem man sich um jeden Datenpunkt einen Kegel mit dem Öffnungswinkel τ vorstellt. Daten, deren Kegel sich gegenseitig überlappen, gehören zu einem Cluster τ ist der Testwinkel, der abhängig ist von der Belegungsdichte, angegeben in Vielfachen der Gleichverteilung.

Von den Daten eines Clusters werden die statistischen Parameter berechnet. Signifikante Trennung ist gegeben, wenn sich die Kegel aus sphärischem Öffnungsgrad + Vertrauenskegel nicht mit anderen Clustern überlappen. Sich überlappende Cluster werden werden zu einer Obergruppe zusammengefaßt. Für diese werden die statistischen Parameter erneut berechnet.
