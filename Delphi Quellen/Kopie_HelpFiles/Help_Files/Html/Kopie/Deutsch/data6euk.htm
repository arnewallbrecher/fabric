Manuelle Dateneingabe

Wenn Sie geographische Koordinaten verwenden, müssen Sie das Format der Positions-Daten angeben. Wenn Sie  Gauss-Krueger oder UTM-Koordinaten, verwenden, sind die Koordinaten  Rechtswert und Hochwert.

Diese Daten können entweder mit dem GPS gemessen oder aus einer Karte ermittelt werden.

Die Messung muß mit  'p' für Flächen oder 'l' für Lineare bezeichnet werden.
