Programm ‚Berechnungen‚:

Es können folgende Berechnungen durchgeführt werden:

Berechnung der Schnittkante von 2 Flächen,

Berechnung des Winkels zwischen 2 Flächen,

Berechnung des Winkels zwischen 2 Linearen,

Berechnung einer Fläche aus 2 Linearen,

Berechnung der Normalen auf 2 Lineare.

Auf dieser Startseite muß im Menu ‚Dateneingabe‚ das Laufwerk ausgesucht werden, in dem die Daten stehen. Im Menu ‚Berechnungen‚ wird bestimmt, was ausgerechnet werden soll. Hat man manuelle Eingabe gewählt, so muß noch eingegeben werden, ob die Winkel in Altgrad oder Neugrad gemessen wurden. Für Dateneingabe von einem Datenträger entfällt diese Angabe, da bei der Datenspeicherung alle Daten in Altgrad gespeichert werden.

Die Auswahl aus den Menus wird in der unteren Zeile dargestellt. Wenn alle nötigen Menus angeklickt wurden, ist die Taste „weiter“ aktiv, mit ihr gelangt man zum folgenden Formular, entweder zur manuellen Dateneingabe oder zur Eingabe von einem Datenträger.
