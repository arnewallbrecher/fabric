Eingabe von Versatzdaten aus einer Datei

Nach Drücken von 'Start' zeigt das Listenfeld auf der rechten Seite  ('Dateien wählen') die Dateien des Ordners, dessen Pfad  in Fabric.ini gespeichert ist, die die Erweiterung '*.SLI' haben. Wenn dieses Feld leer ist und die Anzeige  'keine Datei gefunden' erscheint, kann der Pfad nach Drücken von  Pfad finden' geändert werden.
Klicken Sie auf die Dateien, die Sie benutzen wollen. Diese erscheinen im mittleren Feld ('Gewählte Dateien'). Dateien können wieder von dieser Liste entfernt werden, indem man sie anklickt..
Die Taste  'Auswahl  ok' beendet die Wahl der Dateien.
die Daten der gewählten Dateien erscheinen im linken Listenfeld.
Die Extension der gewählten Dateien ('*.Sli') erscheint unter der linken Liste.
Die Taste  'Plot' öffnet das Grafik-Formular.
