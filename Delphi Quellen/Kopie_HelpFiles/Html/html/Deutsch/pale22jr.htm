Striemungs-Lineation

Eine Striemungs-Lineation ist ein Linear auf einer Störungsfläche, das durch die Bewegung des Hangendblocks erzeugt wurde. Solch eine Fläche wird Harnischfläche genannt. Striemungs-Lineationen markieren die Richtung des 'maximum resolved shear stress' (MRSS). Sie haben eine Polarität.'
