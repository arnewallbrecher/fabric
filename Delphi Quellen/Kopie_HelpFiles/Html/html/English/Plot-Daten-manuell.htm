Manual Data Input:

The form for manual data input is similar to those of the previous programs. The input of azimuth of dip and angle of dip is separeted by 'enter'. The input of a fabric element is terminated by 'OK' or 'enter'. The typed value is shown in the list field. Input is terminated by 'terminate data input'. This activates the menus 'Save data', 'Printer', and the button 'plot'. The 'plot' button opens the plot form. Maximum number of data shows the value which is written in Fabric.ini.
