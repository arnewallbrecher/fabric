Statistical Measures of Circular Distributions:

With this program the statistical measures of circular orientation data (i.e. strike lines fotolineations, flow directions) are calculated. These data can either be pure directions without defined magnitude ( strike lines ) or they can have defined lengths ( fotolineations). In the menu 'data‘ the choice between pure directions and directions with lengths must be made.

The menu ‚range‘ differentiates between data without polarity (strike lines) and those with polarity ( flow directions).
The first type of data uses only the range 1..180°. The second one needs the range 1..360°.

For manual data input the choice between 1..360° and 1..400gon must be made in the menu 'angle'

The extensions for files of these data types are:

*.STR no polarity, pure directions
*.LST no polarity, directions with lengths
*.AZI with polarity, pure directions
*.LAZ with polarity, directions with lengths
