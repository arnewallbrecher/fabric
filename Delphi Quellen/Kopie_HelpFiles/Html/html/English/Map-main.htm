Plot Data into a Map

This program plots measurements (linears or planes) onto a map. Data can be entered in the following ways:

1)manual input
2)from a file
3)from a sample map
4)from an Excel file

The program needs the coordinates of the data points (geographic coordinates as dd°mm.mmm',or dd°mm'ss.s''), Gauss-Krueger coordinates or UTM-Coordinates), and azimut and dip angle of the measures, together with a symbol for linears or planes.
Measures can be entered using a 0..360° or 0..400gon scale. Input from a file is always with 0..360°.
For manual input and input from a map, the sectors (NE, SE, SW, NW) are needed to find the geographic coordinates. NE means northern latitude and eastern longitude, the other sectors are used in a similar sense.
