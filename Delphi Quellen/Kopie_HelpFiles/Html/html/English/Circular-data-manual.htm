Manual Input of Circular Distributed Data:

Input of azimuth and length (only for directions with lengths) is terminated with ‚enter‚ or ‚OK‚. The data appear in the list field. Press the button 'terminate data input' when input is finished. The menus 'save data' and 'printer' become active. The button 'measures' starts the calculation.

Maximum number of data shows the value which is set in Fabric.ini.
