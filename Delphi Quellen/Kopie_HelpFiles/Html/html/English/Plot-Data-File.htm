Data Input from a File

After pressing the button 'Start' the list field on the right side ('select files') shows the content of the directory whose path is saved in Fabric.ini. If this field is empty and the message 'no file found' appears, the data path can be changed after pressing the button 'find data path'.
Click the files you want to use. These files appear in the middle field ('selected files'). Files can be removed from this list by clicking on them.
The extension of the files and the path chosen appear under the left list.
The button 'plot' opens the plot form. the data of the used files appear in the left field.
