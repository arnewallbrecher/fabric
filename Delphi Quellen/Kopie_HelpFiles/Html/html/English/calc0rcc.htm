Manual Data Input:

The orientation of planes or linears is either described by azimuth of dip and angle of dip or by strike, angle of dip and direction of dip. You have to fill in the orientations of the two fabric elements which shall be used in the calculation. Clicking  'OK' or pressing the return key starts the calculation. Input data and calculated fabric elements appear in the list field.

After pressing the 'terminate input' button, the input data and/or the results can be saved or printed.

Maximum number of data indicates the number of data sets which can be calculated.
You can change this number, if necessary.
