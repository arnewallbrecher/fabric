Input from a File

Data are saved in files with the extension '*.map' as geographic coordinates Gauss-Krueger or UTM coordinates, and azimuths of dip und dip angle. All files with extension '*.map' of the chosen folder appear in the right list field (The folder is changed with the button 'change directory'). Click the file with which you want to draw a map. You may also click more than one file, if you want to draw a map for several files. The choice of files is terminated with the button 'selection ok'. The data of all chosen files appear in the left list field. With the button 'plot' you reach the graphic screen which displays the map.
