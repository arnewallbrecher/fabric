Extensions:

Fabric7 uses the following extensions for file names:

*.AZI      = horizontally distributed orientation data with polarity, 0° to 359°, type: azimuth,
*.BMP   = Windos Bitmap Format for graphic files,
*.DXF   =  Drawing Exchange Format for graphic files,
*.EMF   = Enhanced Metafile (Windows) for graphic files,
*.FOU    = Fourier-koefficients, type: vergency,x1,y1 x2,y2 x3,y3,
*.FRN    = Data file for the enhanced Fry method,
*.FRY    = Data file for the Fry method,
*.GIF     = Graphic Interchange Format for graphic files,
*.JPG     = Jpeg format for compressed graphic files,
*.LAZ    = horizantally distributed orientation data with polarity and magnitudes,
*.LIN      = three-dimensionally distributed linear orientation data, type: azimuth of dip / angle of dip,
*.LI2      = Files with pairs of lineations for the calculation of planes, angles, and normals,
*.LST     = horizontally distributed bipolar orientation data with magnitudes, type: fotolineation,
*.PCH   =  Planes (dip azimuth/dip angle) and clockwise Pitch angle,
*.PLA    = three-dimensionally distributed planar orientation data, type: azimuth of dip / angle of dip,
*.PLB    = three-dimensionally distributed planar orientation data, type: strike,dip, direction of dip,
*.PLI      = Planes (dip azimuth/dip angle) and lineation (dip azimuth/dip angle) in the plane,
*.PLT    =  Hewlett Packard Graphic Language for vectorgraphic,
*.PL2     = Files with pairs of planar data for calculation of intersection lines or angles,
*.POL    = X-y-coordinates of Corner points of polygons for the Panozzo method,
*.RFI     = Data for the Rf/Phi method,
*.RGD    = Data for R%-G%-diagramms, type:  R%,G%,
*.SLI      = Slickensides with striations with polarity, type: azimuth of dip /+ or - angle of dip,
*.STR    = horizontally distributed bipolar orientation data, type: strike lines,
*.TAL    = distances of isogons for the T’alpha/alpha method,
*.TIF     =  Tagged Image Format for graphic files,
*.TLI     = Linears (stretching or compression) for the Talbot method,
*.WMF  = Windows Metafile for graphic files,
*.WOO  = Files with eigenvalues, type: E1,E2,E3.
