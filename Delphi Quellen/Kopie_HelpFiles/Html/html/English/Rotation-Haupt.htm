Rotation:

This program carries out rotations of orientation data (azimuth of dip/ angle of dip or strike/angle of dip/direction of dip) around an arbitrarily oriented rotation axis. The calculation is done with the use of a rotation tensor.(WALLBRECHER, 1986:24ff).

The menu 'data input' is the same as in the program 'calculations. Additionally input can be made from an excel sheet. Manual input is also possible for universal stage data azimuth and tilt angle of axes (i.e. quartz-c-axes) or normals to planes (i.e. poles to miaca-planes). These data are converted to azimuths of dip and angles of dip internally. All further processing of these data is done in this format. The type of data must be set in the menu 'fabric type'. In the menu ‚angle‚ you choose input in the 1..360° or 1..400gon scale.

For data input from a file, the fabric types "U-stage-axes" and "U-stage-poles“ are not active, because U-stage data are saved in *.LIN files.
