Small Circles, Manual Data Input:

The form for manual data input is similar to that of the previous programs. The input of azimuth of dip and angle of dip of the axes of the small circles is separated by 'enter'. The input of the apical angle is finished by 'OK' or 'enter'. The typed data are shown in the list field. Input is terminated by 'terminate data input'. This activates the menus 'Save data', 'Printer', and the button 'plot'. The 'plot' button opens the plot form. Maximum number of data shows the value which is written in Fabric.ini.
