Fabric:

This program plots structural orientation data in various projections and calculates the statistical parameters. The methods and algorithms are described in detail in WALLBRECHER (1986).

In the menu 'data input' you choose between manual data input, input from a file, or input from an Excel sheet.
Manual input is also possible for universal stage data like azimuths and tilt angles of axes (i.e. quartz-c-axes) or normals to planes (i.e. poles to mica-planes). These data are internally converted to azimuths of dip and angles of dip. All further processing of these data is done in this format.
The type of data must be set in the menu 'fabric type'. In the menu ‚angle‚ you choose input in the 1..360° or 1..400gon scale.
For data input from a file the fabric types "U-stage-axes" and "U-stage-poles“ are not active, because U-stage data are saved in *.LIN files.
For data input from an EXCEL-sheet  azimutth of dip and angle of dip must either be written in a single row (with a separation key) or in two separate rows.
