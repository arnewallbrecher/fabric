Program 'Calculations':

In the menu 'data input' you choose the type of input, either manually or from a file. In the menu 'calculations'‚ you choose the type of calculation.

For manual input you also have to choose the scale in which the angles have been measured (1..360° or 1..400gon). For data input from a file this information is not necessary, because all saved data have the 1..360° scale.

The chosen options are shown in the line at the bottom. When all options have been chosen, the button 'continue' becomes active. It leads to the next form.
