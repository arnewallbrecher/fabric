Vielfache der Gleichverteilung

Eine Gleichverteilung liegt vor, wenn auf gleichgroßen Flächen der Lagenkugel gleichviel Daten liegen. Teilt man die Fläche der Halbkugel durch die Anzahl der Daten, so erhält man die Fläche, auf der jeweils ein Messwert liegen würde, wenn Gleichverteilung vorläge. Mehrere Daten auf dieser Fläche ergeben ein Vielfaches der Gleichverteilung.
