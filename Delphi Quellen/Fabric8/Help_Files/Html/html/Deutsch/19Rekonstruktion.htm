Rekonstruktion einer Datenliste aus einem Diagramm

Importieren Sie ein Diagramm, klicken Sie im Menu "Projektion" auf die "Projektionsmethode", mit der das Diagram erstellt wurde. Im Menu "Gefügetyp" wählen Sie den Typ der dargestellten Daten. Klicken Sie anschließend auf die Endpunkte des EW-Durchmessers des Diagramms. Wenn Sie nun einen Punkt in dem Diagramm anklicken, erscheint sein Wert mit Fallazimut und Fallwinkel in der Liste. Wenn alle Punkter angeklickt sind, drücken Sie die Tasre "Rekonstruktion beenden". Jetzt können die Daten gespeichert und/oder gedruckt werden.
