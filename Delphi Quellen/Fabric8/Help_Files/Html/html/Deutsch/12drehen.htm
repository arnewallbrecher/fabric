drehen (Falten, Fourier-Analyse)

Um eine Fourier-Analyse durchzuführen, muß die Falte symmetrisch sein, d.h. eine geneigte und/oder vergente Falte muß vor der Analyse zurückgeformt werden. Wenn die Faltenumbiegungen nicht in einer horizontalen Ebene liegen, wird die Falte rotiert, um sie in diese Lage zu bringen. Hierzu wird eine Umbiegung angeklickt. Nach Anklicken der zweiten Umbiegung rotiert das Programm die Falte.
