Die errechneten b1- und b3-Koeffizienten werden in das Diagramm geplottet.Die Punkte sind numeriert Die Farbe kann mit dem entsprechenden Menu geändert werden. Das abgegrenzte Feld repräsentiert den Bereich der möglichen Faltenformen. In der Liste auf der rechten Seite werden die berechneten Koeffizienten b1, b2 und b3 ausgegeben. die mit Buchstaben bezeichneten Linien bedeuten:

F = Zickzackfalten,
D = parabelförmige Falten,
C = halbellipsenförmige Falten,
A = Kofferfalten.

Die Zahlen in den rotumrandeten Kreisen bedeuten zunehmende Amplituden (HUDDLESTON, 1973).
