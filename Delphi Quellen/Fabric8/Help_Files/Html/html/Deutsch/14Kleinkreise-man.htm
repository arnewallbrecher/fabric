Kleinkreise, manuelle Dateneingabe:

Das Formular für die manuell Eingabe entspricht den vorigen Programmen. Jede Eingabe wird mit "Enter" abgeschlossen. Die eingegebenen Werte werden im Listenfeld angezeigt. Die Eingabe wird mit "Dateneingabe beendet“. Mit der Taste "Plotten" wird das Grafikformular geöffnet. max Datenmenge‚ zeigt die in Fabric.ini eingestellte maximale Datenmenge an, dieser Wert kann auf der Startseite geändert werden.
