Extension

Fabric8 verwendet folgende Extensionen für Dateinamen:

        *.AZI     =  Flächig verteilte Richtungsdaten 0 bis 359 Grad Typ Azimut,
          *.BMP   =  Windows bitmap für Graphik-Dateien,
        *.DXF   =  Drawing Exchange Format für Autocad-Vektorgraphik,
        *.EMF   =  Enhanced Windows Metafile für Graphik-Dateien,
        * FLI     =  Daten für ein Flinn oder Hsü-Diagramm,
          *.FOU     =  Fourier-Koeffizienten Typ Vergenz,x1,y1 x2,y2 x3,y3,
        *.FRN   =  Dateien für die normierte Fry-Methode
        *.FRY   =  Dateien für die Fry-Methode,
        *.GIF    =  Graphic interchange format für Graphik-Dateien,
        *.JPG    =  JPeg-Graphik-Dateien mit Datenkompression,
        *.LAZ   =  Polare flächig verteilte Richtungsdaten mit Beträgen,
        *.LIN    =  Räumlich verteilte lineare Richtungsdaten, Typ Fallazimut/Fallwinkel,
        *.LI2     =  Dateien mit Paaren von Lineationen für die Berechnung von Flächen, Winkeln oder Normalen,
        *.LST   =  Bipolare flächig verteilte Richtungsdaten mit Beträgen, Typ Fotolineation,
        *.MAP  =  Datei einer Strukturkarte (Koordinaten und Messwerte),
        *.PCH  =  Flächen (Fallazimut/Fallwinkel) und Pitchwinkel im Uhrzeigersinn,
         *.PLA  =  Räumlich verteilte flächige Richtungsdaten, Typ Fallazimut/Fallwinkel,
        *.PLB    = Räumlich verteilte flächigeRichtungsdaten,  Typ Streichen,Fallen,Fallrichtung,
        *.PLI    = Flächen (Fallazimut/Fallwinkel) und Linear (Fallazimut/Fallwinkel) in der Fläche (Pitch),
        *.PL2   = Dateien mit Paaren von Flächendaten zur Berechnung von Schnittlinearen,,
        *.PLT  =  Hewlett Peckard Graphic Language tum Plot von Vektor-Graphiken,
        *.POL  =  X-Y-Koordinaten von Polygon-Eckpukten für die Fry- und Panozzo-Methode,
        *.RFI   = Daten für die Rf/Phi-Methode,
        *.RGD  = Daten für R%-G%-Diagramme Typ R%,G%,
        *.SCI   =  Daten von Kleinkreisen,
        *.SLI    = Harnischflächen mit polaren Striemungsrichtungen, Typ Fallazimut/Fallwinkel/Fallazimut/+  od. - Fallwinkel,
        *.STR  = Flächig verteilte bipolare Richtungsdaten, Typ Streichlinien,
        *.TAL  = Isogonenabstände für T’alpha/Alpha,
        *.TLI   = Daten für die Talbot-Methode,
        *.WOO  =   Eigenwerte für ein Woodcock- oder Vollmer-Diagramm.
