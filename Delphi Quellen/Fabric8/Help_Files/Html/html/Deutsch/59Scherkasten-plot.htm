Simulation ebener Verformung, Plot

Deformiert werden können entweder auf die Plotfläche gezeichnete Objekte (Linien, Rechtecke, Ellipsen) oder Bitmaps. Das Bitmap kann entweder aus der Zwischenablage oder aus einer Datei geladen werden. Es muß das Format *.bmp verwendet werden.
Die Deformation kann eine Scherung oder eine Plättung sein. Scherung wird in Gamma (= Tangens des Scherwinkels) angegeben, Plättung in Prozent auf die die vertikale Kürzung erfolgen soll. Die Scherung kann affin oder nicht affin sein. Bei nicht affiner Deformation wird der Faktor angegeben, um den die Deformation nach oben zunehmen soll.
