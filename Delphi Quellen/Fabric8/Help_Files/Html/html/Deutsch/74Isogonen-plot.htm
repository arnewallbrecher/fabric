Isogonen zeichnen

Das Menu "Bild importieren" ermöglicht das Importieren einer Bitmap (*.BMP, *.GIF, *.TIF, *.JPG, *.WMF oder *.EMF) aus einer Datei oder aus der Zwischenablage.
Mit dem Menu "Bild korrigieren" kann das Bild so gedreht werden, daß die Wendepunkte einer gefalteten Lage in einer horizontalen Linie liegen. Hierzu muß eventuell der Mittelpunkt der Rotation neu definiert werden: Klicken Sie auf die Option 'neuer Mittelpunkt' im Menu "Bild korrigieren". Klicken Sie mit der linken Maustaste auf den neuen Mittelpunkt und drücken Sie die Taste "Neuer Mittelpunkt". Das Bild wird nun verschoben. Zum Rotieren Wählen Sie die Option "Wendelinie horizontieren". Klicken Sie mit der linken Maustaste auf den ersten Wendepunkt und drücken Sie die Taste "1. Punkt". Klicken Sie auf den zweiten Wendepunkt der gleichen Lage und drücken Sie die Taste "2. Punkt".
Drücken Sie die Taste "Isogonen zeichnen" und wählen Sie die Strichstärke aus der Liste. Farbe und Linientyp können im Menu "Linien" gewählt werden. Andere Linientypen als durchgezogene Linien können nur die Strichstärke 1 haben.
Zum Zeichnen der Null-Isogone klicken Sie mit der linken Maustaste doppelt auf die erste Schichtgrenze, halten Sie die Maustaste gedrückt und gehen Sie zur nächsten Schichtgrenze und lassen Sie die Maustaste los. Die Isogone wird nun gezeichnet. Halten Sie die Maustaste wieder gedrückt und gehen Sie zu nächsten Schichtgrenze. Wenn alle möglichen Isogonen gezeichnet sind, drücken Sie die Taste "nächster Kippwinkel" und wiederholen Sie den Vorgang.
Die Grafik kann zu jedem Zeitpunkt gespeichert oder gedruckt werden.
