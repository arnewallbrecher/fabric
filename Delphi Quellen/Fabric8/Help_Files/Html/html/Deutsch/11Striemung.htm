Striemungs-Lineation

Eine Striemungs-Lineation wird durch Bewegung des Hangensblocks auf einer Srörungsfläche erzeugt. Eine Störung mit solcher Striemung wird Harnisch-(Fläche) genannt. Die Striemungs-Lineation markiert die Richtung des 'maximum resolved shear stress (MRSS)'. Striemungs-Lineationen besitzen eine Polarität.
