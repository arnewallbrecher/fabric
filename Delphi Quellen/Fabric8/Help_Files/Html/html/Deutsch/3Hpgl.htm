In eine HPGL-Datei zeichnen

Mit dieser Option kann eine *.plt-Datei erzeugt werden, mit der eine Vektorgrafik erstellt werden kann. Eine solche Datei kann nach CorelDraw und anderen Programmen importiert werden. Wenn diese Option angeklickt ist, erscheint rechts neben der Grafik das Panel "Plot", in dem der Radius des Diagramms und die Position des Mittelpunktes gewählt werden müssen. Die Eingaben der letzten Fabric8-Sitzung sind gespeichert und können übernommen werden. Nach klicken auf 'OK' wird die Datei erstellt. Sie wird in den Ordner geschrieben, dessen Pfad in Fabric.ini eingetragen ist.
