Flinn- und Hsü-Diagramm, Plot

Flinn-Diagramm (Flinn, 1962):
Auf der Ordinate wird das Verhältnis a = X / Y dargestellt und auf der Abzisse das Verhältnis b = Y / Z. Der k-Wert                                                

                                                     k = (a-1) / (b – 1) 

beschreibt die Form eines Ellipsoides. k = ∞ ergibt sich bei einem prolaten (uniaxiale Längung) Ellipsoid (Abb. 1). Prolate Formen liegen auf der Ordinate.
k = 0 ergibt sich aus einem oblaten (uniaxiale Kürzung) Ellipsoid (Abb. 2). Oblate Formen liegen auf der Abszisse. k = 1 ergibt sich bei ebener Verformung (Abb.3). Bei dieser Deformation bleibt die Y-Achse des Ellipsoids unverändert.    

 



Hsü (Nadai)-Diagramm::
Dies Diagramm verwendet den Lode Parameter (n) anstelle des k-Wertes.

                                                                  
eist die natürliche Verformung (ln(1+e). n liegt zwischen -1 (prolat) und +1 (oblat). n = 0 steht für ebene Verformung. Auf dem Lode Parameter wird die Intensität der Deformation, abgeleitet von der Arbeit, die nötig ist, um eine Kugel (es = 0) zum Ellipsoid (es > 0) zu deformieren, abgetragen. iese errechnet sich nach dem Verformungs-Intensitätsfaktor
