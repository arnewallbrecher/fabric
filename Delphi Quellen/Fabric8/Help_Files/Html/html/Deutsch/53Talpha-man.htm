Talpha'/Alpha-Methode, manuelle Dateneingabe

Für jede Falte wird zunächst der Scheitelabstand zwischen zwei Flächen eingegeben, dann die Isogonen-Abstände für alle erreichbaren Isogonen in Zehnerschritten. Eingegebene Daten erscheinen in dem rechten Listenfeld.
Die Daten können in einer Datei mit der Extension ‚.TAL‘ gespeichert werden. Die Daten-Datei speichert jedoch direkt die Verhältnisse T/T0.
