Plot von Paläospannungen

Die kompressiven Bereiche werden mit  Isolinien in  Lambertscher Projektion dargestellt. Diese Linien trennen Bereiche mit verschiedenen Kompressionsgraden (>20%, >40%, >60%, >80%, > 95%).
Im Menu "Farbe/Isolinien" kann die Farbe der Isolinien oder verschiedene Graustufen gewählt werden.
Im Menu "Konturierung"‚ können weitere Methoden der Darstellung der  Belegungsdichte gewählt werden.
Das Menu "Glättungsgrad" wird nur für die Scalarprodukt-Methode verwendet.
Das Menu "Grafik" ist genauso wie in den anderen Programmen.
Die errechneten Maximal-  und Minimalwerte erlauben eine Abschätzung der Konsistenz eines Datensatzes.
Im Idealfall ist das Maximum gleich der Anzahl der Daten und das Minimum null.
