Strukturgeologische Daten:

Es existieren Programme zur Auswertung von:

Flächenmessungen (Fallazimut/Fallwinkel)

Messungen von Linearen (Fallazimut/Fallwinkel)

Lineationen aus Luft- oder Satellitenbildern

Versatzdaten (Harnischfläche, Striemung, Polarität)

Faltenformen (Isogonen, Fourier-Analyse)

Straindaten (Achsen des Strain-Ellipsoids, Elongationen)
