Datenpunkte aus einer Karte

Das Programm ermittelt die geographischen, Gauss-Krüger- oder UTM-Koordinaten eines in die Karte eingetragenen Meßpunktes. Die Karte muß als *.bmp-, *.gif-, *.tif-, *.jpg-, *.wmf- oder *.emf-Datei vorliegen, bzw. in der Zwischenablage vorhanden sein. Die Kartenränder müssen in geographischen, Gauss-Krüger- oder UTM-Koordinaten in die entsprechenden Felder eingetragen werden. Im Falle, daß die Meßpunkte durchgestochen und auf der Rückseite der Karte markiert wurden, kann die Karte seitenverkehrt dargestellt werden.

Vorgangsweise:
1) Im Menu 'Karte importieren' wird gewählt, ob eine Datei von einem Datenträger oder aus der Zwischenablage importiert werden soll.
2) Im Menu 'Markierung der Lokationen' muß eingegeben werden, ob die Kartenvorder- oder rückseite importiert werden soll.
3) Eintragen der Randkoordinaten der Karte in die entsprechenden Felder.
4) Anklicken eines Meßpunktes
5) Dateneingabe: 'p' oder 'l' für Fläche bzw. Linear, Azimut, Fallwinkel.
6) Wenn alle Punkte angeklickt sind auf 'Dateneingabe beenden' klicken.
7) Mit 'Plot' die Karte erstellen.
