Dateneingabe aus einer Datei

Nach Drücken der Taste "Start" erscheint im rechten Listenfeld ("Dateien auswählen") der Inhalt des Ordners, dessen Pfad in Fabric.ini eingetragen ist. Ist dies Feld leer und der Hinweis 'Keine Daten vorhanden' erscheint, so kann man nach Drücken der Taste "Daten-Pfad suchen" den Pfad neu bestimmen.

Dateien, deren Daten berechnet werden sollen, werden angeklickt, worauf sie im mittleren Listenfeld ("ausgewählte Dateien") erscheinen. Versehentlich gewählte Dateien können durch Anklicken wieder aus der Auswahl entfernt werden.

Mit der Taste "Berechnen" wird die Rechnung eingeleitet. Ausgansdaten und Ergebnisse erscheinen im linken Listenfeld.

Zusätzlich wird die Extension angegeben, die vom Datentyp abhängig ist.

Der aktuelle Daten-Pfad erscheint links unten.
