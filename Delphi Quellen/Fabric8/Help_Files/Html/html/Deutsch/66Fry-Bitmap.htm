Fry-Methode, Bitmap laden

Dieser Programmteil dient der Erstellung einer Koordinaten-Matrix. Nach Klicken auf "Start" erfolgt die Aufforderung, ein Bitmap zu importieren. Hierzu wird das Menu "Bild importieren" angeklickt.
Vorgangsweise für die konventionelle Fry-Methode:
Wenn das Bild auf dem Bildschirm erschienen ist, werden der Reihe nach alle Mittelpunkte der Marker angeklickt, wobei diese nach Augenmaß gewählt werden. die Koordinaten der Punkte erscheinen im Listenfeld auf der rechten Seite.
Vorgangsweise für die normierte Fry-Methode:
Wenn das Bild auf dem Bildschirm erschienen ist, wird ein Mittelpunkt eines Markers, der nach Augenmaß gewählt wird, angeklickt, dann ein Punkt auf dem Rand des Markers. Das Programm berechnet aus diesen zwei Punkten den Radius des Markers. Der Vorgang wird wiederholt, bis alle Marker angeklickt sind. Die Koordinaten und die Radien erscheinen in dem rechten Listenfeld.
Die Farbe der Punkte kann im Menu "Farbe" geändert werden.
Wenn alle Punkte angeklickt sind, wird die Taste "Punkteingabe beenden" gedrückt. Hierdurch wird das Menu "Punkte speichern" aktiv, mit dem die Koordinaten der Mittelpunkte gespeichert werden.
Mit der Taste "Plotten" wird die Grafik erzeugt.
