Rf/Phi: Dateneingabe aus einem Bild

Die deformierten Gerölle oder andere Marker müssen in einer Pixelgrafik (*.bmp, *.gif, *.tif, *.jpg, *.wmf, *.emf) abgebildet sein oder als Grafik in der Zwischenablage gespeichert sein. Klicken Sie im Menu "Bild importieren" entweder auf 'einfügen' (Zwischenablage) oder auf 'Aus einer Datei' und dann auf das entsprechende Bildformat  Es erscheint eine Liste mit allen  gespeicherten Bildern im entsprechenden Format. Klicken Sie auf das Bild, das Sie bearbeiten wollen.
Klicken Sie der Reihe nach zuerst auf die Endpunkte der langen Achsen der deformierten Objekte, dann auf die kurzen. Die letzte Eingebe kann mit der Taste "Letzte Ellipse löschen" wieder gelöscht werden . Nach Drücken der Taste "Eingabe beenden" können die Daten gespeichert werden. Mit der Taste 'Plotten" gelangt man zur Auswertung.
