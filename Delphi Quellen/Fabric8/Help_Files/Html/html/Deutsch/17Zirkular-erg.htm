Maßzahlen zirkularer Verteilungen, Ergebnisse

Die verwendeten Dateien werden aufgelistet und der Winkelraum,den die Daten einnehmen, und die Anzahl der Daten wird angezeigt. Die statistischen Maßzahlen der Verteilung sind:

  • Die Vektorsumme,
  • der normierte Summenvektor,
  • der prozentuale Regelungsgrad,
  • das Vektormittel,
  • die Schiefe der Verteilung,
  • die Kurtosis (Wölbung) der Verteilung

Die Maßzahlen werden für eine zirkulare Normalverteilung d.h. für eine von-Mises-Verteilung (von MISES, 1918) berechnet, sie sind ausführlich bei WALLBRECHER, 1986:124ff beschrieben.
