Talpha'/Alpha-Methode

Dies Programm plottet die Längen von Isogonen einer Falte normiert auf den Scheitelabstand zweier Schichtgrenzen gegen die Fallwinkel der Isogonen. Konzentrische Falten ergeben eine horizontale Linie, Kongruente Falten ergeben eine Kosinus-Funktion (RAMSAY, 1967).
Die Dateneingabe erfolgt entweder manuell oder aus einer Datei oder grafisch aus einer *.bmp, *.gif, *.tif, *.jpg, *.wmf oder *.emf-Datei.
