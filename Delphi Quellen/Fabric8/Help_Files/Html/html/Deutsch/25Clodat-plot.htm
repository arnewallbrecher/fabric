Cluster von Richtungsdaten, Plot

Die signifikante Trennung von Verteilungshäufungen (Clustern) wird vorgenommen.
Für die gefundenen Cluster werden die statistischen Parameter wie in Programm 8 berechnet. Spärischer Öffnungsgrad und Vertrauenskegel werden um den Cluster-Schwerpunkt (Schwerpunktvektor)gezeichnet. Im Menu "Konfidenz" kann das in Fabric.ini gespeicherte Vertrauensintervall geändert werden.

Im Menu "Farbe" können die in Fabric.ini gespeicherten Farben aller graphisch dargestellten Parameter geändert werden. Im Menu ‚Datenpunkte‚ kann das Symbol für die Datenpunkte geändert werden.

Das Menu "Graphik" ist das gleiche wie bei den vorigen Programmen.
