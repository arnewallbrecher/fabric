Plot eines Hoeppener-Diagramms

Die Versatz-Daten werden als Hoeppener-Diagramm geplottet.Die Projektion und die Farben der Flächenpole und der Pfeile können in den entsprechenden Menus geändert werden.

Wird im Menu "überarbeiten" 'ja' gewählt, so kann man interaktiv Daten entfernen, die nicht in den Datensatz passen.
