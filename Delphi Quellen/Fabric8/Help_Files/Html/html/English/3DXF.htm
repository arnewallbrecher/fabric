Plot into a DXF-File:

This option produces a *.dxf-file which is used to draw a vector plot. Such a file can be imported into AutoCad, CorelDraw and other plot programs.
This option opens a panel on the right hand of the plot surface for input of the radius of the diagram and the coordinates of the centre. The readings from the last Fabric8 application are saved in Fabric.ini and displayed. They can be changed. With "OK" the file is produced.
