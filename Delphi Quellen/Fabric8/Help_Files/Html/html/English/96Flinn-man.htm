Flinn and Hsü diagram, manual data input

Input of axes:

Each input of an axis must be followed by 'return'. The input values are shown in a list on the right side. Only integer numbers are accepted.

Input of elongations:
The elongations e1, e2, e3 (e = (deformed length – undeformed length) / undeformed length) are measured parallel  to the principal axes X > Y > Z of the strain ellipsoid and given in integer numbers. e3 is always  negative.

 For both type of data input is terminated with the button 'Terminate input'. Now the menus  'Save data' and 'Print' become active. Both data types are saved as axest. (Exemple: The data set e1 = 40%, e2 = 20%, e3 = -20% is saved as 140, 120, 80). The button e 'Plot' opens the plot form.
