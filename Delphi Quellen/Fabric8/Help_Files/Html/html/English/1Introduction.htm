Introduction to Fabric8:

Fabric8 is a program package for analysis, plot, and statistical evaluation of structural data . It has been developed by:


Although the program has been running for a long time without errors, we cannot exclude that an error might occur in a certain unexpected combination of data. If this is the case, we shall send you a corrected version free of charge. We are also thankful for suggestions for improvement.

The program set consists of the executable program FABRIC8.exe and it's Dll-codes,  and the files LICENCE, DEUTSCH ENGLISH, HILFDEUT, and HELPENGL.

The file FABRIC.ini saves all the settings from the last use of Fabric8. The file TESTCONFIG, is used to check whether the lines of Fabric8 are complete and the spelling is correct.

On the start page the menu language offers the choice between German and English. Clicking on manual opens the explanations. With the button 'Start' you reach the list of applications.

Attention: If you installed Fabric8 on a system with read only restriction, you have to install the folder FabricIni to a drive which allows writing. Do not change or extend the name 'FabricIni.
