Manual Data Input

The orientation of planes or linears are measured as azimuth of dip and angle of dip. Input is the orientation of the fabric elements which are used for calculation. The calculation starts after clicking after filling the input field and pressing the return key. Initial data and the calculated fabric element appear in a list.
After pressing the button 'Terminate input' initial data and results can be saved and/or printed.
The maximum number of data is the amount of data for which the program is set. This number can be changed on the first form of every program. Changes are saved in 'Fabric.ini'
