Rotation, Data Input from a File

After pressing the button "Start" the list field on the right side ("Select files") shows the files with extension '*.lin' or '*.pla' of the  folder whose path is saved in Fabric.ini. If this field is empty and the message "No file found" appears, the data path can be changed after pressing the button "find data path".
Click on the files you want to use. These files appear in the middle field ("Selected files"). Files can be removed from this list by clicking on them.
Now you have to fill in the orientation (azimuth of dip / angle of dip) of the rotation axis and the rotation angle (clockwise, looking in the direction of dip of the rotation axis) The rotation starts after pressing the button "Calculate". The data and their rotated values appear in the left list field.
The extension of the files and the path chosen appear under the left list.
