Plot of a Rose Diagram

The circular orientation data are plotted in a histogramm. Three types of diagrams can be drawn:

1.) Block diagrams
2.) Star diagrams
3.) Diagrams with floating intervals

The interval describes the width of the classes, i.e. the sector angle, in which the data are put together.
For floating intervals, the inkrement describes the angle of overlap of classes or sectors. It should be a devisor of the interval angle.
The menu "Fill the diagram" decides whether the diagram will be filled with the chosen colour or not.
In the menu "Colour" the drawing colour is chosen.
All settings will be saved in the file Fabric.ini.
