Strain Analysis with the Talbot Method, Manual Data Input

The manual input is the same as for orientation data (Azimuth of dip / angle of dip or strike, angle of dip, dip direction). Compressional structures are written with negative azimuths of dip, or with negative strike. The input of azimuth of dip and angle of dip is separated by "Enter". Input of a fabric element is terminated by "Enter. The input is shown in the table on the right side. The button "Terminate input" activates the menus "Save data" and "Print". The button "Plot" opens the plot form.
