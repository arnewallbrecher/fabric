Input of Circular Data from a File

After pressing the button "Start", the list field on the right side ("Select files") shows the files with the relavant extension of the folder whose path is saved in Fabric.ini. If this field is empty and the message "No file found" appears, the data path can be changed after pressing the button "Find data path"'.
Click on the files you want to use. These files appear in the middle field ("Selected files"). Files can be removed from this list by clicking on them.
The data of the used files appear in the left field.
The extension of the files and the chosen path appear under the left list.
Program 6: The button "Measures" opens the form with the table of results.
Program 7: The button "Plot" opens the form for the plot of a rose diagram.
