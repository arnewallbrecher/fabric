Rotation, Manual Data Input:

After pressing the "Start" button, the orientation (azimuth of dip / angle of dip) of the rotation axis is put in. The rotation angle is filled clockwise. (looking in the direction of dip). At this point you fill in the data you want to rotate. The input of each fabric element is terminated by "Enter". The measures and their rotated values appear in the list field. Data input is terminated by pressing the button "Terminate data input". Now the menus "Save data", "Save rotated data", and "Printer2 become active.
