Draw Isogons

The menu "Import picture" imports a bitmap (*.bmp, *.gif, *.tif, *.jpg, *.wmf, *.emf) from a file or from the clipboard.
With the menu "Adjust picture" the picture can be rotated in a way that the inflection points of a folded layer lie in a horizontal line. In order to do so, the centre of the rotation may be redefined. Click on the option 'New centre' in the menu "Adjust picture". Click with the left mouse button on the new centre, then press the button "New centre". The picture will be shifted. Choose the option "Median line horizontal".to rotate the picture. Click with the left mouse button on an inflection point and press the button "Point 1". Click on another inflection point of the same layer and press the button "Point 2".
Press the button "Draw isogons" and choose the pen width from the list. Colour and line type can be set in the menu "Lines". Line types other than solid can only have a pen width of 1.
Double click with the left button on the first layer border, hold the mouse button down and go to the other border of the layer. With left mouse button up the isogon will be drawn. With left mouse button down go to the next border. When all possible isogons are drawn, press the button "Next dip angle" and repeat the procedure.
The graphic can be saved or printed at any time.
