Types of Rose Diagrams

The circular orientation data are plotted in a diagram. For this purpose the circle is divided into sectors. The number of data contained in each sector is counted. The sectors are drawn with radii which are proportional to the number of data in them. The sectors are normalized in a way that the vector which contains the maximum of data equals the radius of the circle.

Three types of diagram can be plotted::

1.) Block roses with the borders of sectors(borders of classes),
2.) Star roses in which the centres of the sectors (centres of classes) are connected by lines,
3.) Diagrams with floating intervals.
