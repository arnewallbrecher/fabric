Plot of Data

The settings of the menus  "Projection“, "Symbol“ und "Colour“ were saved in Fabric.ini. They can be changed interactively.
The Menu "Diagram“ contains an option to save the plot within the frame as bitmap, an option for overlay of the actual plot with another plot, produced by Fabric8. The options 'Plot with a printer', 'Draw into an HPGL file', and 'Plot into a  DXF file' produce vectorgraphics. These options open a  Panel at the bottom of the right side, where you can set the plot parameters. These are saved in Fabric.ini. Further options chose the colour of the diagram, the frame, and the background.
With the option 'Printer' the plot is printed as bitmap.
