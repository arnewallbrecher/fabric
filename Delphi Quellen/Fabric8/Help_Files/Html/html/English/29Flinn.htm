Flinn  Diagram (Flinn, 1962):


The y-axis is the ratio a = X / Y, and the x-axis is b = Y / Z. The k-value

                                                   

                                                     k = (a-1) / (b – 1)

 

describes the shape of an ellipsoid. k = ∞ represents a prolate (uniaxial extension) ellipsoid (Fig. 7). Prolate shapes plot on the y-axis. k = 0 represents an oblate (uniaxial shortening) ellipsoid (Fig. 8). Oblate shapes plot on the x-axis. k = 1 represents plane strain (Fig.9). In this deformation the Y-axis of the ellipsoid is unchanged.
