Direct Method

 This program needs the directions of the principal stresses which you have to find first  with program 11. It is also recommended to check the consistency of the data set with program 13, and to eliminate non fitting data.
The data are rotated into a position with s1 vertical and s3 pointing east. This gives us the stress tensor

                                                               

 In this coordinate system the shape factor for each slickenside and the striation on it  is calculated with Bott's Formula. These values are displayed in a histogram.
