Manual Input of Displacement Data

The form for manual input of slickenside planes and striations is similar to the previous programs. After clicking on the "Start" button, you put in the azimuth and dip of the slickenside plane, followed by azimuth and dip of the striation. The dip angle is positive if the plane is a reverse fault, and negative if the plane is a normal fault. For positive dip angles you may drop the '+' sign.
The input of a fabric element is terminated by "Enter". The input is shown in the list field. Measurement errors which produce an offset between the striation and the slickenside plane, are corrected automatically.
Input is terminated by "Terminate data input". This activates the menus "Save data", "Printer", and the button "Plot". The "Plot"' button opens the plot form. Maximum number of data shows the value which is written in Fabric.ini.
