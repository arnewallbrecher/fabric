Flinn Diagram (Flinn, 1962)

In a xy-coordinate system the ratio a = X / Y is is drawn on tha y-axis and  b = Y / Z on the x-axis. The value  k                                               

                                                     k = (a-1) / (b – 1) 

describes the shape of an ellipsoid. k = ∞ a an extreme prolate (uniaxial extension) ellipsoid (Fig. 7). Prolate shapes are clos to the y-axis.
k = 0 describe an extreme oblate (uniaxial flattening) ellipsoid (Fig. 8). Oblate shape are close to the x-axis. k = 1 indicates plane strain (Fig.9). In this deforamrtion  the intermediate axis of the ellipsoid is unchanged.
