R% - G% - Diagram, Manual Data Input

The degree of preferred orientation and the cylindricity are written in their fields. For each pair of data you have to choose the symbol which shall be plotted from the scroll list "Symbol". Confirm with "ok". The input is shown in the list field on the right side. The maximum number of data can be changed in the file Fabric.ini.
The button "Finish input" activates the menus "Save data" and "Print".
The button "Plot" opens the plot area.
