Test Angle:

In the program "Significant separation of submaxima" the test angle is the apical angle of the cone which is drawn around each data point in order to find those data which belong to a cluster. It is dependent on the size of the area for a multiple of even distribution. It is defined as

                                                       

n = number of data
a = multiple of even distribution
The multiple of even distribution is saved in Fabric.ini. It can be changed in the scroll field on the right side of the plot area.
